1. Add additional components

Switch to the Designer.

Add another YandexTranslator component, and another Label for the second translation.

Rename the Label to TranslationLabel2, and change the Text property to "translated text".

2. Code translation to a different language

Click on Blocks to switch to the Blocks Editor.

Duplicate the YandexTranslate1.RequestTranslation block, then change YandexTranslate1 to YandexTranslate2 so the translator gets called twice.

Then, change the Text block for languageToTranslateTo to "en-it" so it translates to Italian. Or look at the available language codes and try a different one.

Now Duplicate the YandexTranslate1.GotTranslation block and change YandexTranslate1 to YandexTranslate2.

And change TranslationLabel to TranslationLabel2 in the inner block so the second translation will be displayed in the second label.

3. Test it out

Try it out on your app! You should see two translations now!
