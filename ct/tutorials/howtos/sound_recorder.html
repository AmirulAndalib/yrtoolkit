1. Add New Components

Switch to the Designer.

Add a HorizontalArrangement to the Viewer, and drop 3 Buttons into the HorizontalArrangement.

Name them RecordButton, StopRecordingButton, and PlayRecordingButton, in that order and change the Text property for each to its purpose.

Uncheck the Enabled box for the StopRecordingButton and the PlayRecordingButton.

Drag in a SoundRecorder component and another Player component from the Media drawer. Rename the Player component RecordingPlayer.

2. Code the RecordButton

Switch to the Blocks Editor.

Code the when RecordButton.Click event block.

  • Enable the StopRecordingButton so the user can stop the recording when they wish.
  • Disable the RecordButton so the user cannot record while already recording.
  • Disable the PlayRecordingButton so the user cannot play back until finished recording.
  • Start the SoundRecorder.

3. Code the Stop Recording Button

When the user stops recording, you want to:

  • Disable StopRecordingButton.
  • Enable RecordButton.
  • Stop the SoundRecorder1.

4. After Recording

When SoundRecorder1 stops recording, it triggers an event, when SoundRecording.AfterSoundRecorded.

  • Set the Source for RecordingPlayer to the sound returned by the event.
  • And enable the PlayRecordingButton so the user can play it back.

5. Play the Recording

Code when PlayRecordingButton.Click to simply start RecordingPlayer..

5. Test your App

Try out your changes! Press the RecordButton, then press several notes to play some music and record it. When finished, press the StopRecordingButton. Then press PlayRecordingButton to play back your music. How does it sound?
